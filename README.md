# Airflow DAGs

This repository contains Apache Airflow DAGs created for practice and learning purposes as part of the course:

__Building ETL and Data Pipelines with Bash, Airflow, and Kafka__

### Purpose

The goal of this repository is to practice:

- Designing ETL pipelines using Apache Airflow

- Orchestrating tasks with BashOperator and PythonOperator

- Understanding task dependencies, scheduling, and retries

- Working with files, transformations, and data workflows
